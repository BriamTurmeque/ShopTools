dentro de esta carpeta se realizara un ejemplo acerca de la comprobacion de entrada, para verificar si 
algun dato ingresado por el usuario es lo que le estamos pidiendo, por ejemplo si le estamos pidiendo 
un dato tipo numerico, y el usuario ingresa un dato tipo string, esto me va a mostrar un error y lo va a 
obligar a que digite lo que se le esta pidiendo.

Los beneficios de este validacion de datos es que si el usario ingresa un tipo de dato erroneo 
el programa no va a generar error y va a seguir corriendo.
